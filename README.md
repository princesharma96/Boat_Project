# Boat Management System (SQL Project)

## Project Description

This project is a Boat Management System developed using SQL.
It is used to manage boat details, customer information, and booking records in a database.

## Technologies Used

* SQL
* Database Management System (MySQL / Oracle / SQL Server)

## Project File

* SQL_BOAT_PROJECTS.sql – Contains all SQL queries for creating tables and inserting data.

## Features

* Store boat details
* Manage customer information
* Track bookings
* Maintain boat records
